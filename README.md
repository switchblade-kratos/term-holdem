# term-holdem
A CLI for playing a terminal-based Texas Hold'em with your own computer

## Usage

term-holdem [options]

Options:
  -h, --help        Show help
  -v, --version     Show version information

# Installation

git clone https://github.com/<your-username>/term-holdem.git
cd term-holdem
make
sudo make install

# Uninstall 

sudo make uninstall
