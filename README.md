# term-holdem
A CLI for playing a terminal-based Texas Hold'em with your own computer

## Usage

terholdem [options]

Options:
  -h, --help        Show help
  -v, --version     Show version information

# Installation

git clone https://github.com/<your-username>/terholdem.git
cd terholdem
make
sudo make install

# Uninstall 

sudo make uninstall
